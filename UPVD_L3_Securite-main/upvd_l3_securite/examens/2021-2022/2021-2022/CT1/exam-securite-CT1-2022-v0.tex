\begin{exercice}[Virus bash par entrelacement]
%// Virus script bash par entrelacement (faire un mini exemple)

Le but de cet exercice est d'écrire le  code en script bash d'un virus qui va disséminer son code dans le code hote de la manière suivante:
\begin{center}
\includegraphics[width=7cm]{entrelacement} 
\end{center}

\begin{enumerate}
\item Ecrire le  un script bash \texttt{entrelacement.sh} qui prend deux fichiers (\texttt{cible.sh} et \texttt{codeviral.sh}) en argument et qui fabrique un troisième fichier \texttt{resultat.sh} comme suit:
\begin{itemize}
\item si le nombre de ligne de \texttt{cible.sh} est inférieur à \texttt{codeviral.sh} alors ne rien faire.
\item si le nombre de ligne de \texttt{cible.sh} est supérieur à \texttt{codeviral.sh} alors définir 
  $$s=\left\lfloor\frac{\mbox{nombre de ligne de \texttt{cible.sh}}}{\mbox{nombre de ligne de \texttt{codeviral.sh}}}\right\rfloor.$$
  Rappel : $\lfloor x \rfloor = $ partie entière de $x$.
\item remplir le fichier \texttt{resultat.sh} en ajoutant alternativement $s$ lignes de  \texttt{cible.sh} suivies d'une ligne de \texttt{codeviral.sh}.
\end{itemize}

%\noindent\textit{Indication.} Vous pourrez utiliser la boucle suivante
%\begin{verbatim}
%while read ligne
%do
%
%done < $file
%\end{verbatim}
%qui lit le fichier \texttt{\$file} ligne par ligne.

\item Faire un script  \texttt{restauration.sh} qui prend en argument  le nombre \verb|s| et  un fichier \texttt{resultat.sh} fabriqué dans la question précédente. Le script  restaure les  deux fichiers (\texttt{cible.sh} et \texttt{codeviral.sh}). 
\medskip



\item Combiner les deux scripts \texttt{entrelacement.sh} et \texttt{restauration.sh} pour fabriquer votre virus qui infectera uniquement le répertoire courant:
  \begin{itemize}
  \item Le code de restauration restaure le code hote et le code viral dans deux fichiers temporaires, il exécute d'abord le code viral, puis il exécute le code hote.
  \item Le code viral parcourt les fichiers \verb|*.sh| du répertoire courant et ajoute son code par entrelacement comme dans le question 1).
    \end{itemize}
\item Expliquez une stratégie possible pour rendre la segmentation dans la partie \texttt{entrelacement.sh} aléatoire : les nombre de lignes du code hôte consécutive ne sont plus fixes (égaux à \verb|s|) mais varient de façon aléatoire.
\end{enumerate}
\end{exercice}
