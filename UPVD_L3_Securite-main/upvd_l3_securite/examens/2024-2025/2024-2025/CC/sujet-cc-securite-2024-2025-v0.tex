\begin{exercice}
 \begin{enumerate}
 \item On considère le réseau local ci-dessous qui  contient une 50-aines de machines connectées au réseau via un switch et un routeur. L'adresse réseau est 177.12.11.0/24.
   \begin{center}
\includegraphics[width=8cm]{reseau-empoisonement.pdf}
     \end{center}

Un utilisateur malveillant souhaite rendre inaccessible un servuer distant S. Ce serveur n'accepte qu'une 100aines de connexions  TCP simultanées provenant d'adresse différentes. Le serveur a une protection contre l'attaque SYN flooding utilisant : une connexion est ouverte uniquement lors de la reception de l'ACK du client. Proposez une stratégie pour que l'attaquant puisse rendre inopérant le serveur distant. 

\item On considère le réseau ci-dessous interconnecté par des pares-feu. 
  Dans la Zone2 il y a un serveur web (131.159.20.1) sur TCP (ports 80 et 443) et un serveur de mail (131.159.20.2) sur TCP port 25. Les utilisateurs du réseau se trouve dans la Zone1.

   \begin{center}
\includegraphics[width=12cm]{parefeu.pdf}
     \end{center}
On souhaite mettre en place la politique suivante :
\begin{itemize}
\item Les utilisateurs peuvent accéder librement à des services Web, sur le sports 80 et 443, qui se trouvent n'importent où, seulement si c'est eux qui initient la la connexion. Aucune machine extérieur à Zone1 peut initier une connexion avec une machine de la Zone1.

\item Les utiliseurs (Zone1) peuvent accéder aux serveurs de la Zone2 via SSH. Ils peuvent aussi utiliser SSH (port 22) pour  communiquer avec une machine se trouvant sur le \textit{Dangereux Internet}. Cependant, les machines de la Zone2 ne peuvent être contactées sur le port 22 que  par des machines de la Zone1.

\item Tout le monde (inclus le Méchant Internet) peux accéder aux serveur web  et mail  dans la Zone2. Le serveur Web de la Zone 2 doit être accessible en TCP que sur les ports 80 et 443 et le serveur de mail ne doit etre accessible que sur le port 25.

\item Seul le serveur de mail peut initier des connexions vers des serveurs mail situés sur le \textit{Dangereux internet}. 


\end{itemize}
\begin{enumerate}
\item Donnez les règles du pare-feu \textbf{sans état} pour assurer les règles de sécurité ci-dessus.
  \item Donnez les règles du pare-feu \textbf{avec état} pour assurer les règles de sécurité ci-dessus.
\end{enumerate}
\end{enumerate}
 \end{exercice}

\begin{exercice}
  L'objectif de l'exercice est de faire un virus par entrelacement de code caractère par caractère. Vous pouvez utiliser les fichiers fic.txt et quizz.sh pour tester vos scripts. 

  \begin{enumerate}
  \item On considère le code suivant qui affiche une chaîne de caractères mot à mot:
\begin{terminal}
\begin{verbatim}
  ligne="une petite phrase"
  for mot in $ligne
  do
    echo "$mot"
  done
\end{verbatim}
\end{terminal}

modifier ce script afin d'inverser l'ordre de lettres dans chaque mot. Vous n'utiliserez que les commandes \verb|echo|, \verb|wc|,\verb|head| et \verb|tail| pour manipuler les mots.

\item Faire un script \verb|entrelacement.sh| qui prend en paramètre un fichier \verb|fic|, une chaîne de caractères \verb|str|, et un nom de fichier \verb|fic_out|. Ce script extrait les caractères de \verb|str| un à un et les ajoute un à un à la fin des mots de \verb|fic| et les écrits dans \verb|fic_out|.  Vous avez un exemple de résultat d'exécution ci-dessous.
\begin{terminal}
\begin{verbatim}
clegrand@pc-home:~/virus-cc$ cat fic
hello it is a nice day
the sky is blue
and the sun is rising
clegrand@pc-home:~/virus-cc$ ./entrelacement.sh fic "abcdefghijkl" fic_out
fichier=fic, chaine="abcdefghijkl", output=fic_out
clegrand@pc-home:~/virus-cc$ cat fic_out 
helloa itb isc ad nicee dayf 
theg skyh isi bluej 
andk thel sun is rising 
\end{verbatim}
\end{terminal}

  
\item Faire un script \verb|restauration.sh| qui prend en paramètre un fichier \verb|fic_out| produit avec le script \verb|entrelacement.sh|, un nombre de caractères à extraire \verb|n|, et le nom du fichier de restauration de \verb|fic| \verb|fic_rest|. Le script produit le fichier initial \verb|fic| et affiche la chaine  \verb|str|. 

\item Combinez les script  \verb|entrelacement.sh| et \verb|restauration.sh| pour faire un virus par entrelacement. Les cibles du virus sont les fichiers \verb|.sh| exécutable du répertoire courant. 
 \item Proposez une méthode de randomisation de la méthode propagation de ce virus.
  \end{enumerate}
\end{exercice}
